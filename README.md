# Social hooks
A React hook library for social media.


## Installation
```
npm i social-hooks
yarn add social-hooks
pnpm add social-hooks
```


## Summary
- `useSocialAssets`
- `useSocialShare`


## Supported
- Facebook
- X (formerly Twitter)
- Telegram
- Whatsapp
- LinkedIn
- Pinterest
- VK
- Odnoklassniki
- Reddit
- Tumblr
- Mail.Ru
